AI 교육 커리큘럼 자동 생성기
=======================

교육주제와 교육생 정보를 입력하면 AI가 시간별 모듈 구성의 커리큘럼을 자동 생성하는 도구입니다.

주요 기능:
- 교육주제, 교육생, 교육시간(2~8시간) 입력
- 고객 소통 내용(이메일/메모) 붙여넣기로 요구사항 반영
- Claude Haiku 4.5 API로 과정명/카피/목표/기대효과/모듈 자동 생성
- 모듈별 세부내용 6개씩 구성
- PDF 저장 (브라우저 인쇄), HTML 다운로드, 텍스트 복사
- 5단계 프로그레스 애니메이션

기술:
- 프론트엔드: HTML/CSS/JS (단일 파일)
- AI: Anthropic Claude Haiku 4.5 API
- 배포: GitHub Pages

제작: 김종혁 (나눔경영컨설팅)
